# react-boardgame
A single player fantasy board game.

<h3>Planned Features:</h3>

* Login.
* Game log.
* ~~CSS/HTML Gameboard.~~
* Harvest resources.
* Non Player AI.
* Currency.
* ~~Player choose game piece.~~
* Roll 12 sided die.
* Player pieces move automatically.
* Most spaces are purchaseable.
* Chance cards will be given after landing on specific spaces.
